# python-pyqt5-venv

- MacOS(M2 chip)

## Clone

```
$ git clone https://github.com/choewy/python-pyqt-venv
```

## PyQt5

- nodejs
- python3.9

```
$ cd pyqt5
$ npm i
$ npm run install:python
$ npm run venv
$ npm run install:package
$ npm run start
```

## PyQt6

- nodejs
- python3.10

```
$ cd pyqt6
$ npm i
$ npm run venv
$ npm run install:package
$ npm run start
```
